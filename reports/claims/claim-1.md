I want to understand if there is a difference in size between habitable and none habitable planets.

In this claim i decided to make two groups for comparison : 

Group A : Size of habitable planets
Group B : Size of non habitable planets

My Hypothesis : 
H0 : There is no difference in size between habitable and none habitable planets

H1 : There is a difference

To test my hypothesis in this case, i'll be using permutation test. Especially because we have only 13 habitable planets over more than 3500 non habitable


By the end of my test, i got a p-value of 0.0028. 

Since the p_value is less than 0.05, it suggest that it is unlikely that the nulll hypothesis (H0) may be true

However, the number of habitable planets is very small, which may affect the result.
